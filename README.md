## ⚙️ Set-up with TailwindCSS

### Clone the repo

```
git clone https://github.com/NirobSaha420/NS-Workflow.git .
```

### TailwindCSS Installation

```
npm install tailwindcss @tailwindcss/cli
```

```
npm run tw:dev
```

**Note:** `"tw:dev": "npx @tailwindcss/cli -i ./css/input.css -o ./css/output.css --watch"`
